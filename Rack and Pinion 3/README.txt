Summary:
This is a general purpose linear servo actuator (pusher style).  Two sizes have been designed, for different space constraints and force outputs.These use continuous rotation servos which helps keep the cost very low.  Off the shelf actuators of this type can cost around $70 USD.The “mini” version will fit in smaller spacers, but has a much lower force output.
